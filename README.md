# LiDAR Cone Detection using DBSCAN  


## 🌌 Overview  
Ever wondered how self-driving cars understand the world around them?  

This project explores the perception system of autonomous vehicles using **LiDAR data** to detect and isolate objects — specifically cones on a racetrack.  

We convert raw 3D point cloud data into meaningful insights using the **DBSCAN clustering algorithm**, enabling a vehicle to interpret its surroundings.


## 🧠 What This Project Does  
- Processes raw **LiDAR point cloud data**  
- Applies **DBSCAN (Density-Based Clustering)**  
- Identifies and separates **individual cones**  
- Converts noisy sensor data into structured perception  


## ⚙️ Tech Stack  
- Python  
- NumPy  
- Matplotlib  
- Scikit-learn (DBSCAN)  
- LiDAR Point Cloud Data  

## 🧩 Core Concepts  
- Computer Vision  
- Robotics Perception  
- Unsupervised Learning  
- Autonomous Navigation

## 🛠️ How It Works  

1. LiDAR Data Input  
2. Preprocessing (cleaning and structuring points)  
3. Clustering using DBSCAN  
4. Each cluster represents a cone  
5. Visualization of detected cones  


## 📸 Output Visualization  

- Each cone is detected as a separate cluster  
- Noise is automatically ignored  
- Clean segmentation of track elements
